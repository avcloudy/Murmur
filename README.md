# Murmuration
v0.1.0-alpha

Murmur is a Swift UI torrent client, written to broaden my understanding of Swift and Swift UI. Also want to write a phone app manager for the client that runs on iOS and pairs with main app.

### Current Functionality
* UI mockup with example torrents in list

### Working on
* Reimplementing bencode interpretation
* Adding real torrents to list

### Installation
If you really want to, open it in Xcode and let it build. Don't, though.

### Prerequisites
* MacOS
* Swift 6.2
